# Playtime-Monitor

* **A lightweight tool** that monitors active processes and tracks playtime for specific games or applications. 
* **Reward System:** This tool features advanced options such as setting specific games or programs as "saviors." This means completing productive tasks can earn you extra time in categories with limits, encouraging self-discipline or habit-building.
* **Highly Customizable:** You can set specific hours, apps, and categories. For example, you can configure the tool to grant 1 hour of "unwanted activity" for every 2 hours spent on something more important.
* **Anti-Cheat Systems:** Includes multiple prevention systems to stop users from idling in the background to gain time. The tool tracks AFK status and window focus to ensure only active time is counted.
* **Versatile Tracking:** Perfect for keeping an eye on how much time you spend in-game, even for non-Steam titles or apps that don't track time natively.
* **Note on Security:** This tool is not built with high-level security in mind and may have vulnerabilities. It is designed to work exclusively within a local network.

# How to use
1. Download the required `agent.py` and the folders (`templates`, `python_local`, `jumpscares`) and place them all in one directory.
2. Launch the agent through the command line using:  
   `.\python_local\python.exe agent4.py`
3. Access the **Admin Dashboard** via your browser at `http://localhost:5055/`.
4. To use it for **Parental Controls** or remote monitoring, replace `localhost` with the local IP address of the target PC.
5. The program will automatically detect and track the **active processes** you have specified.
6. View your accumulated **playtime logs** and statistics directly through the web interface.
7. All data is saved locally, allowing you to track progress over multiple sessions.
